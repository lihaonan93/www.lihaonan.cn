```bash
nohup script.sh > script.log 2>&1 &
```
1、`script.log`为日志保存输出的文件名称；

2、`2>&1` 表示不仅命令行正常地输出保存到`script.log`中，产生错误信息的输出也保存到`script.log`中；

3、`&`表示该进程在后台运行；

4、`nohup`表示进程在当用户注销(logout)或网络断开时不会被终止

::: tip
注意：若不指定输出日志文件，默认会输出到`nohup.out`文件中.
:::
