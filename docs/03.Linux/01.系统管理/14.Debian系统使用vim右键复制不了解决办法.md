1. 编辑 vim 的默认配置文件
```bash
vim /usr/share/vim/vim82/defaults.vim
```
2. 转至第 82 行，找到：
```bash
set mouse=a
```
3. 修改成：
```bash
set mouse-=a
```
4. 保存即可生效
