##### 递归设置目录ACL权限
```bash
setfacl -m user:alex:rwx -R /NAS 
setfacl -m group:alex:rwx -R /NAS 
```
##### 设置默认ACL权限  
所有新建的子文件都会继承父目录的ACL权限
`setfacl -m d:user:alex:rwx -R /NAS`
##### 删除目录所有ACL权限  
`setfacl -b /NAS/`
