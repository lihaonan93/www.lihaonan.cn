##### grep的排除选项为 `-v`
##### 排除空行的命令是：
`grep -v '^$' filename`
##### 排除以#注释的命令是：
`grep -v '^#' filename`
##### 查找多个关键词，匹配任意一个都输出
> 选项  `-E` 扩展查找模式，与`egrep`命令等效
结合起来就是，既排除空行又排除注释行的命令

```
grep  -Ev  '^$|^#'  filename
egrep  -v  '^$|^#'  filename
```
