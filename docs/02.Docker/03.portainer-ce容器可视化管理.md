#### 拉取镜像

`docker pull portainer/portainer-ce:2.17.1`

#### 启动容器

```bash
docker run -d -p 8000:8000 -p 9443:9443 \
--name portainer --restart=always \
-v /var/run/docker.sock:/var/run/docker.sock \
-v ./portainer_data:/data \
portainer/portainer-ce:2.17.1
```

#### 访问地址

`https://你的域名:9443`
